# Interface SampleBrowsingSession

Instances of this interface are used for browsing samples, including access to all filter columns and the result column as shown in the 'Samples' tab of Bitwig Studio's contextual browser window.

## Method Details

### `getFileTypeFilter`

```java
BrowserFilterColumn getFileTypeFilter()
```

Returns the file type filter as shown in the category column of the browser.

**Returns:** the requested file type filter object.
**Since:** API version 1
