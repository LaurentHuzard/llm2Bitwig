# Interface Groove

An interface representing the global groove settings of the project.

## Method Details

### `getEnabled`

```java
Parameter getEnabled()
```

Returns the enabled state of the groove.

**Returns:** an object that provides access to the groove on/off setting
**Since:** API version 1

---

### `getShuffleAmount`

```java
Parameter getShuffleAmount()
```

Returns the object that represents the shuffle amount in Bitwig Studio.

**Returns:** an ranged value object that provides access to the shuffle amount
**Since:** API version 1
