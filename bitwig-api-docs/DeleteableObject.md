# Interface DeleteableObject

Interface implemented by objects that can be deleted from the project.

## Method Details

### `deleteObject`

```java
void deleteObject()
```

Deletes this object from the document. If you want to delete multiple objects at once, see Host.deleteObjects().

**Since:** API version 10

---

### `deleteObjectAction`

```java
HardwareActionBindable deleteObjectAction()
```

Deletes this object from the document.

**Since:** API version 15

---
